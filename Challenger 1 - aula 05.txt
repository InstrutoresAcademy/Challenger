Challenger 1 - aula 05
*Análise de Mesadas com Pandas e Matplotlib*

1. Introdução ao Projeto:
- o que são bibliotecas
- o que é Pandas e Matplotlib.
- finalidade das bibliotecas: Análise e visualização de dados.
- explicação sobre a coleta de dados (mesadas) e como elas serão utilizadas no projeto.

2. Coleta de Dados:
- como usar a função input() para solicitar ao usuário a entrada dos valores de mesada.
- armazenamento dos valores em uma lista.
- criação de um loop para automatizar a coleta de dados para cada mês.

3. Projeto: Análise de Mesadas com Python
Objetivo do Projeto: Desenvolver uma aplicação em Python que coleta dados de mesadas ao longo de 6 meses, calcula a média das mesadas e gera gráficos para visualizar as variações.
***Conversa sobre o que os alunos fazem com a mesada ou dinheiro ganho.
***Conversa sobre poupar pensando no futuro (investimento).
Parte 1: Configuração do Ambiente
- instalação das bibliotecas necessárias (pandas, matplotlib).
- configuração do ambiente de desenvolvimento (VS Code ou outro editor de código).
Parte 2: Coleta e Organização dos Dados
- implementação do código para coletar os valores das mesadas via input().
- criação de um DataFrame com Pandas para organizar os dados.
- exibição do DataFrame no terminal.
Parte 3: Cálculo da Média das Mesadas
- implementação da função para calcular a média das mesadas utilizando Pandas.
- exibição da média calculada no terminal.
Parte 4: Criação de Gráficos
- implementação de gráficos de linha com Matplotlib para visualizar a variação das mesadas ao longo dos meses.
- personalização do gráfico com títulos, rótulos e marcações.
